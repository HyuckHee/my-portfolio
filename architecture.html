기술적 강조 포인트
⚡
분산 크롤 파이프라인
Scheduler → Bull Queue → Processor 3단 분리. 서버 재시작 시에도 Redis에 작업 보존. 스토어별 서킷브레이커로 장애 자동 격리.
📈
append-only 가격 이력
price_records는 INSERT만 허용. 가격 변동 시에만 저장해 중복 없이 시계열 추적. 30일 히스토리 차트, 알림 비교 가능.
🌐
서버/클라이언트 혼합
상품 목록·상세는 서버 컴포넌트로 SEO 확보. 가격 표시만 클라이언트 컴포넌트로 분리해 환율 토글 즉시 반영.
🛡️
타입 안전 풀스택
Drizzle ORM으로 DB 스키마 → TS 타입 자동 생성. Zod로 환경변수 런타임 검증. 잘못된 설정 시 서버 시작 자체 차단.
